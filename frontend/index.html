JLPT Grammar API

A simple read-only API for retrieving JLPT grammar points.

Base URL

Depending on where you are accessing the API:

  • Local development: http://localhost:4000
  • Deployed on Vercel: https://jlpt-grammar-api.vercel.app

All endpoints are relative to the base URL, e.g., /api/grammar/N5 or /api/grammar/N4.

Endpoints

Get All Grammar by Level

GET /api/grammar/:level

Retrieves a JSON array of all grammar points for a specific JLPT level. Supported levels: N5 and N4.

URL Parameters

  • :level (string) - Required. The JLPT level (e.g., N5, N4).

✅ Success Response (200 OK)

Returns a full array of grammar objects.

[{
  "id": 1,
  "grammar_point": "ちゃいけない・じゃいけない",
  "romaji": "cha ikenai / ja ikenai",
  // ...
}]

❌ Error Response (404 Not Found)

Returned if the requested level is not found.

{
  "message": "Error fetching data for level N3. Level not found or is empty."
}

Get Single Grammar by ID

GET /api/grammar/:level/:id

Retrieves a single grammar point object specified by its :level and :id.

URL Parameters

  • :level (string) - Required. JLPT level (e.g., N5, N4).
  • :id (integer) - Required. ID of the grammar point to retrieve.

✅ Success Response (200 OK)

Returns the single matching grammar object.

{
  "id": 15,
  "grammar_point": "ほうがいい",
  "romaji": "hou ga ii",
  // ...
}

❌ Error Response (404 Not Found)

Returned if the requested :id does not exist for that level.

{
  "message": "Grammar point not found in N4"
}
